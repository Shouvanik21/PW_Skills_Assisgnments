1.What is a computer?
Ans:- A computer is an electronic device that stores and processes data often in binary form in accordance with commands given by the user.

2.What is RAM?
Ans:-RAM(Random Access Memory) is a primary memory that stores data in it while some task takes place. It is volatile in nature and is fast since it is based on semiconductor technology. 

3.Where is data stored in a computer?
Ans:-In a computer the data is temporarily stored in the RAM.

4.What is that input device used to type text and numbers on a document in the computer system?
Ans:-Keyboard is thee device.

5.What are the output devices?
Ans:-Those devices that give the result of some data is known as output devices.
Eg:- Monitor, Printer etc.
6.Which is the input device that allows a user to move the cursor or pointer on the screen?
Ans:- Mouse allows the user to move the pointer all around the screen.

7.Which language is directly understood by the computer without a translation program?
Ans:-Machine language or binary language is directly understood by the computer.

8.What are Input devices?
Ans:-Input devices are the devices that take data input from the user for processing.
